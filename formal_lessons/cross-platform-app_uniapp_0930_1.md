### 双线程架构

为什么要用双线程

```html
<body></body>
<script></script>
```

1. 下载 h5，下载 js，js/h5 异步 -- 单线程的问题，慢。

2. web 是不是有很大的安全风险
   - JavaScript 去调用 window.api

- 需要一个沙箱环境来处理开发者编写的 JavaScript 代码。这个沙箱环境不能有任何 window.api
